Alright, getting to the topic at hand.
This is a technique I learned while doing a course on udemy, very good course, highly recommend it. What I am doing here is, giving position relative to the container div, which allows me to absolutely position the div containing text content on top of it. Next, I give absolute position to the text context. Now positioning the text in center horizontally is relatively straight forward, width of 100% and text-align will do the trick. To position vertically, we need to do these steps: 1) Give top and left values for position, Here we are bringing the top of the text div at the center of the image, left is at 0 (remember the width is 100%, so left is 0) 2) We move the text vertically so that instead of its top being at 50%, its center coincides with the image's center, we are doing that by transform: translateY(-50%); which is basically moving the text div vertically along the negative y-axis 50% relative to the size of the text div.

.large-hero {
  position: relative;
  &__text-content {
   position: absolute;
   top: 50%;
   left: 0px;
   width: 100%;
   text-align: center;
   transform: translateY(-50%);
  }
}
